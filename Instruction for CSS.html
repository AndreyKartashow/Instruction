Instruction for CSS

Основные сведения

CSS - язык стилистики web-сайтов. Он предназначен для оформления сайта и его оборачивания в красивую оболочку. CSS может работать только совместно с HTML, при этом они всегда находятся в разных файлах.
Для подключения CSS к файлу HTML необходимо в блоке head прописать тег 'link', в котором указывается наименование стиля и путь к нему.
Для создания определенного стиля в файле CSS указывается наименование блока, к которому будет применен данный стиль и затем в фигурных скобках указывается наименование атрибута и его значение. В общем случае это может выглядеть так:

Селектор
{
Наименование атрибута : значение атрибута
}

Важно так же отметить, что CSS как и HTML не является языком программирования.


Виды селекторов

Селектор - указатель объекта или логически связанной группы, которым можно задать единый стиль. Различают следующие виды селекторов:

  • наименование блока;

  • наименование класса;

  • наименование ID;

  • блок класса;

  • множественные селекторы;

  • общий селектор;

Для обращении к селектору блока, необходимо просто указать его имя, например body. Обращение к селектору класса осуществляется через точку, например .class. Обращение к селектору ID осуществляется через Шарп (#), например #ID_name. Обращение к блоку класс осуществляется через селектор класса с указанием частного блока, например .class p. Обращение ко множественному селектору осуществляется через перечисление элементов, например .class, #ID_name. Обращение к общему селектору осуществляется через *, например *.


Виды атрибутов

Условно, все атрибуты можно поделить на такие группы как:

  • атрибуты для редактирования текста;

  • атрибуты для задания цвета;

  • атрибуты для установления границ;

  • атрибуты для редактирования моделей;

  • стили шрифтов.

Давайте рассмотрим некоторые из способов задания атрибутов по каждому виду.


Атрибуты для редактирования текста

  • для того чтобы установить размер шрифта, необходимо использовать атрибут 'font-size';

  • для того чтобы установить стиль шрифта, необходимо использовать атрибут 'font-family';

  • для того чтобы выделить шрифт курсивом, необходимо использовать атрибут 'font-style';

  • для того чтобы задать шрифту толщину, необходимо использовать атрибут 'text-weight';

  • для того чтобы выделить текс подчеркиванием или зачеркиванием, необходимо использовать атрибутs 'text-decoration';

  • для того чтобы задать расстояние между символами, необходимо использовать атрибут 'letter-spacing';

  • для того чтобы задать расстояние между словами, необходимо использовать атрибут 'word-spacing';

  • для того чтобы установить высоту строки, необходимо использовать атрибут 'line-height';

Важно запомнить, что все стили, относящиеся к одному роду, можно записывать перечислением. Так же важно знать, ч то приоритетность стилистики всегда направлена к более конкретным блокам.


Атрибуты для задания цвета

  • для того чтобы задать цвет фона, необходимо использовать атрибут 'background-color';

  • для того чтобы задать цвет шрифта, необходимо использовать атрибут 'color';

Важно отметить, что цвет можно указывать по его названию, по его шифру и по коду RGB.


Атрибуты для установления границ

  • для того чтобы установить границу, необходимо использовать атрибут 'border-width';

  • для того чтобы задать стиль границы, необходимо использовать атрибут 'border-style';

  • для того чтобы задать цвет границы, необходимо использовать атрибут 'border-color';

  • для того чтобы скруглить границы, необходимо использовать атрибут 'border-radius';

Важно отметить, что границы можно устанавливать не только по всему периметру блока, но и по конкретным сторонам.


Атрибуты для редактирования моделей

  • для того чтобы задать отступ текста внутри блока, необходимо использовать атрибут 'padding';

  • для того чтобы задать отступ блока, необходимо использовать атрибут 'margin';

  • для того чтобы установить ширину блока, необходимо использовать атрибут 'width';

  • для того чтобы установить максимальную ширину блока, необходимо использовать атрибут 'max-width';

  • для того чтобы разместить блок по центру, необходимо использовать атрибут 'margin-auto';

  • для того чтобы разместить блок по центру, необходимо использовать атрибут 'margin-auto';

  • для того чтобы задать тень блоку, необходимо использовать атрибут 'box-chadow';

  • для того чтобы задать блоку фон в виде картинки, необходимо использовать атрибут 'background-image';

Важно отметить, что границы можно устанавливать не только по всему периметру блока, но и по конкретным сторонам.
