Основные сведения

Язык Markdown - это простой и удобный язык разметки текста, который позволяет быстро и легко создавать структурированные документы с использованием минимального количества символов. Он позволяет добавлять заголовки, списки, цитаты, ссылки, изображения, код и многое другое. Markdown является основным языком для создания README файлов в репозиториях на GitHub и других подобных сервисах. С помощью Markdown вы можете создавать красиво оформленные документы и быстро их распространять в интернете. Он также очень удобен для создания документации, блогов, статей и простых веб-страниц.

Заголовки

Заголовки обозначаются с помощью символа # и расставляются перед текстом. Чем меньше символов # используется, тем меньше будет заголовок.

# Заголовок первого уровня

## Заголовок второго уровня

### Заголовок третьего уровня

Для создания списков можно использовать маркированные (символ * или -) и нумерованные списки.

1. Пункт 1

Форматирование

Вы можете форматировать текст, используя следующие теги:

  • жирный текст - обрамляйте текст двумя символами звездочки (**);
  • курсивный текст - обрамляйте текст одной символом звездочки (*);
  • зачеркнутый текст - обрамляйте текст двумя символами тильды (~~);

Ссылки

Вы можете создавать ссылки на другие страницы или ресурсы с помощью следующего синтаксиса:

[Текст ссылки](адрес ссылки)

[Ссылка на Google](https://www.google.com)

Изображения

Чтобы вставить изображение, используйте следующий синтаксис:

![Текст, который будет отображаться при недоступности изображения](адрес изображения)

![Логотип GitHub](https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png)

Цитаты

Чтобы создать цитату, используйте символ > перед текстом цитаты.

> Это пример цитаты.

Код

Для вставки кода используйте символ (обратный апостроф) перед и после кода. Если нужно вставить блок кода, используйте три символа перед и после блока кода.

`print("Hello, World!")`

`def add_numbers(x, y):`

`return x + y`

Горизонтальная черта

Для создания горизонтальной черты можно использовать символы тире (-), подчеркивание (_) или звездочки (*).

---

Таблицы

Чтобы создать таблицу, разделяйте ячейки вертикальной чертой (|), а первую строку отделите от остальных символом минус (-).

| Заголовок 1 | Заголовок 2 |

| ----------- | ----------- |

| Ячейка 1 | Ячейка 2 |

| Ячейка 3 | Ячейка 4 |

Списки задач

Чтобы создать список задач, используйте символы [ ] для невыполненных задач и [x] для выполненных.

- [x] Выполнить задачу 1

- [ ] Выполнить задачу 2

Сноски

Чтобы добавить сноску, используйте следующий синтаксис:

Этот текст содержит сноску[^1].

[^1]: Сноска содержит дополнительную информацию.

Ссылки на якоря

Вы можете добавить ссылку на якорь на странице с помощью следующего синтаксиса:

[Текст ссылки](#имя-якоря)

[Перейти к следующему разделу](#раздел-2)

Текст на несколько строк

Если вы хотите написать текст на несколько строк, то просто перенесите его на новую строку. Если вы хотите создать новый абзац, то вставьте пустую строку между абзацами.

Это первый абзац.

Это второй абзац.
